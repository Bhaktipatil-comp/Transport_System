# Transport Management System

## Description
This project is designed to manage public transportation efficiently using fundamental Data Structures. It provides a structured system for handling ticket booking, cancellations, passenger records, and route planning in an optimized way.

## Features
- Ticket booking management using Queue (FIFO principle)
- Cancellation handling with history tracking
- Dynamic passenger record management
- Route planning and connectivity management
- Efficient ticket searching mechanism
- Simple and user-friendly console interface

## Tech Used
- C Programming Language
- Data Structures

## Data Structures Used
- Linear Queue for ticket booking
- Circular Queue for cancellation logs
- Linked List for passenger management
- Graph for route planning and network representation
- Binary Search for efficient ticket lookup

## How to Run
1. Download or clone the repository
2. Open the project in any C IDE (such as CodeBlocks, VS Code, or Turbo C)
3. Compile the source code
4. Run the main program file

## Output
This system enables efficient management of transport-related operations including booking, cancellation, passenger data handling, and route management. It demonstrates the practical application of Data Structures in solving real-world problems.

- Implementation of multiple Data Structures in a single system
- Real-world problem solving using C
